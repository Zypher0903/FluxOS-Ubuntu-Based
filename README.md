## Features

- Hyprland with gorgeous animations, blur, rounded corners & gradient borders
- Pre-configured Waybar • Wofi • Dunst • Kitty (Dracula theme)
- Full Serbian keyboard layout (US + RS Latin + Alt+Shift toggle)
- Belgrade timezone & locale ready
- Firefox • LibreOffice • GIMP • VLC • Timeshift • rsync
- Ubiquity installer included
- Live USB ready (persistence coming soon)
